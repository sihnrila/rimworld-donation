# 스트리머 설치 가이드

## 제공 파일

- `RimWorld Donation Events MVP` 모드 폴더
- `server` 연동 프로그램

## 설치 순서

1. RimWorld 모드 설치
   - `rimworld-mod` 빌드 결과를 RimWorld `Mods` 폴더에 넣습니다.
   - 게임 실행 후 모드를 활성화합니다.

2. 연동 프로그램 실행

```bash
cd server
npm install
npm run dev
```

3. 테스트 이벤트 발송

```bash
npm run test:chzzk
```

4. 게임 내 확인
   - 후원 금액에 따라 아이템/동물/습격 이벤트가 실행됩니다.

## 후원 금액 기본 매핑

| 금액 | 이벤트 |
|---:|---|
| 1,000원 이상 | 실버 드랍 |
| 3,000원 이상 | 동물 폭주 |
| 5,000원 이상 | 적 습격 |
| 10,000원 이상 | 강한 습격 |

## 실제 판매 시 설명 문구

후원이 들어오면 림월드 안에서 자동으로 이벤트가 터지는 세팅입니다. 예를 들어 5천 원 후원은 습격, 1만 원 후원은 강한 습격, 3천 원 후원은 동물 이벤트처럼 방송 리액션이 나오게 맞춰드립니다.
